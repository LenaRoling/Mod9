Design Discussion

This website was designed with earthy tones to reflect the peacefulness of houseplant care. The layout uses Bootstrap which not only makes the page respopnsive but helps make the styling easier on the developer. I chose the “Quicksand” font from Google Fonts because it’s modern, clean, and easy to read. I used spacing, padding, and visual alignment to keep the content organized and accessible. The header, nav, and footer are the same across all pages to create a cohesive look. I also added floated images to each content section for visual interest.

Validation Test Results:

All five HTML pages were tested with validator.nu and returned zero errors. "Document checking completed. No errors or warnings to show. Used the HTML parser. Externally specified character encoding was utf-8. Total execution time 188 milliseconds."
